Ultrasonic Radar Project for TM4C123GH6PM (EduBASE)
---------------------------------------------------
Components:
- US-100 sensor (UART mode): PE4/PE5 (UART5)
- SG90 Servo: PB5 (Timer1B PWM)
- Red LED Alert: PF1
- LCD (optional): EduBASE LCD port

Groups to create in Keil:
- Startup
- Main Application
- UART (US100)
- Servo Control
- LCD (optional)
- Utilities (Delay, LED)
